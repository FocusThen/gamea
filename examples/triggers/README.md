# Trigger Examples

This directory contains examples of using the trigger system.

## Examples

### moving-platform.lua
Demonstrates creating a moving platform that moves back and forth continuously.

### scaling-object.lua
Shows how to scale entities (shrink/grow) using triggers.

### sequence-chain.lua
Example of chaining multiple actions together in a sequence.

### timer-trigger.lua
Auto-activating trigger that activates after a delay.

### activate-chain.lua
Trigger that activates another trigger, creating chain reactions.

## Usage

These are reference examples. Copy the configuration to your Tiled map objects.

See [Trigger System Guide](../../docs/guides/trigger-system.md) for detailed documentation.
